## 🌿 Flujo de trabajo (Git)
1. Crea tu rama a partir de `main` actualizado:
```bash
   git checkout main
   git pull origin main
   git checkout -b feature/nombre-de-tu-feature
```
2. Haz commits descriptivos (`feat:`, `fix:`, `docs:`, etc.)
3. Sube tu rama y abre un Pull Request hacia `main` para revisión
